## Descripción / Description

**Español**  
StrongMuscle es una aplicación web desarrollada con Angular y Firebase. El objetivo del proyecto es aplicar los conceptos fundamentales del desarrollo frontend: componentes, servicios, rutas protegidas, manejo del estado global y conexión con una base de datos real.

**Características**
- SPA construida con Angular 21 y Standalone Components
- Autenticación real con Firebase Authentication
- Perfil de usuario guardado en Firestore
- Mapa de calor para el seguimiento mensual de entrenamientos
- Plan de rutinas semanal organizado por grupos musculares
- Rutas protegidas con Guards y versiones bloqueadas para usuarios sin sesión

---

**English**  
StrongMuscle is a web application built with Angular and Firebase. The goal of the project is to apply core frontend development concepts: components, services, protected routes, global state management, and integration with a real database.

**Features**
- SPA built with Angular 21 and Standalone Components
- Real authentication with Firebase Authentication
- User profile stored in Firestore
- Activity heatmap for monthly workout tracking
- Weekly workout plan organized by muscle groups
- Route guards with locked preview pages for unauthenticated users
